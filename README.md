# 🎫 EventFlow — Experiência Premium de Bilheteria

EventFlow é uma plataforma de venda de ingressos de última geração, projetada com foco em uma experiência de usuário imersiva e estética premium. Construída com tecnologias web modernas, oferece uma jornada fluida desde a descoberta do evento até a compra segura do ingresso.

---

## ✨ Principais Funcionalidades

- **🚀 Seção Hero Imersiva**: Descoberta de eventos visualmente impactante com busca dinâmica e eventos em destaque.
- **📍 Seletor de Assentos Interativo**: Sistema de seleção de assentos responsivo e de alto nível, com zonas VIP e Pista com disponibilidade em tempo real.
- **💳 Fluxo de Checkout Dinâmico**: Processo de pagamento seguro em múltiplas etapas, com resumo consolidado do pedido e validações em tempo real.
- **🎭 Animações e Transições**: Alimentado por `Framer Motion` para uma sensação de interface viva, fluida e sofisticada.
- **📱 Totalmente Responsivo**: Otimizado para todos os dispositivos, desde monitores de alta resolução até smartphones.
- **🔒 Experiência Segura**: Indicadores de segurança de compra integrados e simulação de tratamento de dados criptografados.

---

## 🛠️ Stack Tecnológica

- **Core**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Estilização**: Vanilla CSS (Sistema de Design Personalizado)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Ícones**: [Lucide React](https://lucide.dev/)
- **Build**: Compilação de produção otimizada via Vite

---

## 🎨 Sistema de Design

O EventFlow utiliza um sistema de design exclusivo baseado em:
- **Cor Primária**: `#7c3aed` (Violeta Vibrante)
- **Glassmorphism**: Camadas sofisticadas semi-transparentes com desfoque de fundo (backdrop-blur).
- **Tipografia**: Fontes sans-serif modernas e marcantes para um visual editorial de luxo.
- **Micro-interações**: Efeitos sutis de hover e transições de escala para maior engajamento.

## 🚀 Como Começar

### Pré-requisitos
- Node.js (v18+)
- npm ou yarn

### Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/LuSilva710/EventFlow.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
